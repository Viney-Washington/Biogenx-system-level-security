## Overview
This project analyzes system-level security risks and cyber resilience strategies for BioGenX following the Phantom Phoenix Advanced Persistent Threat (APT) attack. The analysis focuses on how attackers progress through the intrusion lifecycle and how organizations can strengthen defenses to reduce impact and maintain operational stability.

## Objectives
- Analyze the Phantom Phoenix APT attack lifecycle  
- Evaluate BioGenX’s cyber resilience effectiveness  
- Develop automated system hardening strategies  
- Organize security outputs into a technical control report  
- Predict future attack risks using threat modeling  

## Key Skills Demonstrated
- Cyber Resilience Strategy  
- Threat Modeling and Risk Analysis  
- System Hardening and Security Automation  
- Vulnerability Management  
- Technical Control Reporting  
- Incident Analysis and APT Lifecycle Understanding  

## Key Analysis Areas

### 1. APT Attack Lifecycle Analysis
- Reconnaissance  
- Initial Compromise  
- Lateral Movement  
- Persistence  
- Data Exfiltration  

### 2. Cyber Resilience Evaluation
- Identified gaps in detection and monitoring  
- Evaluated weaknesses in early-stage threat visibility  
- Assessed response and recovery capabilities  

### 3. System Hardening and Automation
- Developed automated scripts to enforce secure configurations  
- Implemented software validation and execution controls  
- Strengthened patch management and access control enforcement  
- Reduced configuration drift across systems  

### 4. Technical Control Reporting
- Structured reporting using FedRAMP Continuous Monitoring framework  
- Tracked system status, remediation efforts, and compliance  
- Applied POA&M methodology for risk tracking and resolution  

### 5. Future Threat Prediction
- Applied threat modeling to identify potential attack paths  
- Conducted vulnerability assessments to detect emerging risks  
- Identified residual risks such as credential abuse, social engineering, and zero-day vulnerabilities  

## Tools and Frameworks
- NIST Risk Management Framework (RMF)  
- FedRAMP Continuous Monitoring  
- OWASP Threat Modeling Principles  
- Vulnerability Management Practices

This project demonstrates how layered security strategies, automation, and continuous monitoring improve cyber resilience. It highlights the importance of proactive defense, early detection, and structured reporting when responding to advanced persistent threats.
